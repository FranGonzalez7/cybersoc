# CyberSOC Básico - Proyecto UD4

Infraestructura de un CyberSOC básico desplegado con Docker.

## Stack Tecnológico
- **SIEM**: Elastic Stack (Elasticsearch + Kibana)
- **Recolección**: Filebeat
- **Gestión de Incidentes**: TheHive
- **Generación de Tráfico**: Kali Linux

## Instalación

1. Clonar el repositorio
2. Ejecutar: `docker-compose up -d`
3. Acceder a Kibana: http://localhost:5601
4. Acceder a TheHive: http://localhost:9000
